# @neville.dabreo/greetingbot

Removes all spaces from a string.

## Install

```
$ npm install @neville.dabreo/greetingbot
```

## Usage

```js
const greetingbot = require("@neville.dabreo/greetingbot");

greetingbot("Neville!");
//=> "Hello Neville!"

greetingbot();
//=> "Hello good looking!"

```
